## Дизеринг
Курсовой проект по компьютерной графике. Реализация методов дизеринга и анализ результатов при помощи DSSIM и PSNR.

Для запуска требуется:
  * libmagick++-dev
  * Qt 5.7+
  * graphicsmagick-libmagick-dev-compat
  * libwebp-dev
